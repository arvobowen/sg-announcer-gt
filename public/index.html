GitHub Announcer Orb

A SpiderGate orb that listens for GitHub release events and sends rich notifications to Microsoft Teams.

View on GitHub API Docs & Testing

Listens for Releases

Monitors your GitHub repositories for new 'published' release events via a secure webhook.

Rich Notifications

Sends beautifully formatted Adaptive Cards to Microsoft Teams channels with release details.

Stable vs. Pre-Release

Intelligently routes notifications to different channels based on whether a release is stable or a pre-release.
